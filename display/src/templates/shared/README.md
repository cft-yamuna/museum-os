# Shared Templates

Utility templates that don't map to a specific catalog app number.

## Templates
- `ProximityTemplate.tsx` — Sensor-triggered content display (TPL-09)
- `MultiScreenTemplate.tsx` — Multi-screen synchronized exhibit (TPL-10)
- `TouchScrollTemplate.tsx` — Touch-scrollable content display (TPL-07)

## Usage
These templates serve cross-cutting needs:
- **Proximity**: triggers content playback based on sensor distance
- **Multi-Screen**: coordinates content across multiple screens in a single exhibit
- **Touch Scroll**: provides scrollable long-form content display
